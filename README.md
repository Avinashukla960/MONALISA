# MONALISA
A Fashion Clothing Store
Monalisa is a modern and user-friendly eCommerce web application designed to provide a seamless online shopping experience. It allows users to browse products, view detailed descriptions, add items to their cart, and make secure purchases with ease.

The platform focuses on simplicity, performance, and a visually appealing interface, making it suitable for both customers and developers exploring full-stack development.
🚀 Features
🛒 Add to Cart & Remove from Cart
🔍 Product Search & Filtering
📦 Product Detail Pages
👤 User Authentication (Login/Signup)
💳 Secure Checkout System
📱 Responsive Design for all devices
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript (or React if used)
Database: MongoDB
Version Control: Git & GitHub


🎯 Purpose

This project is built to:

Practice full-stack web development
Understand eCommerce workflows
Improve UI/UX design skills
Learn API integration and database handling
